Абстрактный класс! :
 - Может наслдоваться только от одного класса и любого коичества интерфейсов
Абстрактные методы :
 - Абстрактный класс может содержать абстрактные методы. Но может и не содержать их вообще.
Методы с реализацией :
 - Абстрактный класс может содержать методы с реализацией.
Данные :
 - Никаких ограничений.
Создание объекта :
 - Нельзя создать объект абстрактного класса.

Интефейс! :
 Наследование 
 - Интерфейс не может наследоваться от классов, но модет от любого количества интерфейсов.
Абстрактные методы
 - Все не статические и не defailt методы интерфейса - абстрактные  - не содержат реализации. Интерфейс может не содержать никаких методов вообще.
Методы с реализацией :
 - Интерфейс может содержать методы по умолчанию (default methods)
Данные :
 - Интерфейс содержит только public final static данные.
Создание объекта:
 - нельзя создать объект интерфейса.
